> 长久陪伴在岔路女巫身侧的魔宠；对于猎犬来说，她身上有着非同一般的王族气质。

### 解锁

满足以下条件就会开始出现：

+ 至少进行 12 次游戏。

+ 解锁 <InlineLabel key="myFriends" />，至少花费 <InlineLabel key="witchsDelight" num={3}/> 在升级魔宠上。

+ 与赫卡忒在三岔路口交谈，她会告诉我们 “赫库芭与盖尔现在愿意与墨利诺厄
并肩作战了”。在哀悼原野再次见到赫库芭，就可以收服。

招募赫库芭需要满足要求：

+ 招募其它三只魔宠：<InlineLabel key="frinos" />、<InlineLabel key="toula" />、<InlineLabel key="raki" />。

+ 携带 <InlineLabel key="frinos" /> 招募一次迷途暗灵。

+ 携带 <InlineLabel key="raki" /> 开采一次矿藏。

+ 至少花费 <InlineLabel key="witchsDelight" num={4}/> 在升级魔宠上。

### 能力

+ 帮助采集种子。

+ 每隔一段时间击晕周围敌人 1 秒，还会在紧要关头挖出 <InlineLabel key="health" /> 或 <InlineLabel key="magick" />。

+ 额外 +15 <InlineLabel key="maxMagick" />。

+ 额外资源概率 +10%。

### 升级

+ **灵之纽带**：+30 / +45 / +60 / +120 <InlineLabel key="maxMagick" />。

+ **土之纽带**：+13% / +16% / +19% 额外资源概率。

+ **御之纽带**：每隔 偶尔 / 7-8 / 5-6 秒击晕周围敌人 1 秒。
