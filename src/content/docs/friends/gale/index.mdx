> 岔路女巫信赖的魔宠之一；终日神出鬼没，但绝不是羞于见人。

### 解锁

满足以下条件就会开始出现：

+ 至少进行 12 次游戏。

+ 解锁 <InlineLabel key="myFriends" />，至少花费 <InlineLabel key="witchsDelight" num={3}/> 在升级魔宠上。

招募加莉需要满足要求：

+ 招募其它三只魔宠：<InlineLabel key="frinos" />、<InlineLabel key="toula" />、<InlineLabel key="raki" />。

+ 携带 <InlineLabel key="frinos" /> 招募一次迷途暗灵。

+ 携带 <InlineLabel key="raki" /> 开采一次矿藏。

+ 至少花费 <InlineLabel key="witchsDelight" num={4}/> 在升级魔宠上。

加莉会出现在普罗米修斯的门口，站在门的右侧。

### 能力

+ 帮助采花。

+ 额外花朵概率：10%。

+ 闪避概率和移动速度 +3%。

+  挺身而出挡下伤害，然后反击并造成 250 伤害，每个区域 3 次，每次需间隔 30 秒。

### 升级

+ **自然纽带**：增加 3% / 4% / 5% / 6% 的闪避几率和移动速度。

+ **荒野纽带**：提高 10% / 13% / 16% / 19% 花朵出现的概率。

+ **暗幕纽带**：反击时造成 250 / 300 / 350 / 400 点伤害。
