银白之颅原本属于栽决者勒瓦尔，而勒瓦尔如今属于你。

这是勒瓦尔昝由自取，而且恕我直言，你对待她比起她原先对待其他
人要负责得多 ———— 算她走运。

## 基础操作

用爆炸性的攻击炮轰敌人，并用前冲的特技拾取头颅弹药。

+ 攻击

普通攻击：向前发射一个骷髅头，骷髅头到达射程极限会落地开始读条，等待一段时间会飞回角色位置，也可以在读条结束前走到落点位置拾取骷髅头，初始只有 3 个骷髅头弹药，用完无法继续攻击。

+ Ω 攻击

原地蓄力，向指定方向投掷一个骷髅头造成大范围 AOE 伤害。

+ 特技

向指定方向位移一段距离。

+ Ω 特技

原地蓄力向指定方向冲去，途中还会向两边释放光波。
