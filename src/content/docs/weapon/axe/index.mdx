……这样的想法多么令人恼火，不是么？事实上，佐利菲特并不在意是谁在使用她，只要能让她啜饮到鲜血就行。

## 基础操作

用沉稳的攻击摧垮敌人，或用特技放手并反击。

+ 攻击

普通攻击，3  次为一个循环，第 3 次攻击会打出大范围的 AOE，但有明显的前摇。

+ Ω 攻击

原地蓄力，最高使用 20 点法力，挥动斧头向周围转圈。

+ 特技

往指示器方向格挡。

+ Ω 特技

原地蓄力并格挡敌人攻击，消耗 25 点魔力向前方释放 3 个大范围的 AOE 伤害。
