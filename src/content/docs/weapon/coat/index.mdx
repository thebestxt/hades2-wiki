暗夜武器诞生于暗影之中，但孕育它们的暗影未曾消散，而是凝聚成了这一最终形态。

所谓武器，便是我们的躯体与意志的延伸。柯辛斯则是这一点的具象体现。

## 基础操作

+ 攻击

向前方打出连续直拳。

+ Ω 攻击

蓄力期间可以移动，期间可以格挡，松手后打出爆裂直拳。

+ 特技

向前方发出一组导弹。

+ Ω 特技

蓄力展示一个扇形区域，蓄力期间导弹会锁定敌人，松手后会轰炸敌人。
